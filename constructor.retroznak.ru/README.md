# constructor.retroznak.ru

Интерактивный конструктор адресных табличек компании «Ретрознак». Сделан на Next.js 15 с TypeScript и React.

> **Примечание:** Этот проект является частью монорепозитория. См. [общий README](../README.md) для информации о всех проектах.

> **Продакшен:** Проект разворачивается на домен `land.retroznak.ru` (заменяя лендинг). На одном домене работает только один проект.

## Возможности

- Выбор стиля таблички (Ленинградский, Петроградский, Мини)
- Настройка надписей (улица, номер дома)
- Выбор материала (обычная, оцинкованная, нержавеющая сталь)
- Выбор цветов по каталогу RAL
- Дополнительные опции (подсветка, рельеф, фотореле)
- Предварительный просмотр на Canvas в реальном времени
- Отправка заявки на изготовление

## Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build
npm start
```
