# Настройка целей в Яндекс.Метрике

## Обзор

На сайте настроена отправка событий в Яндекс.Метрику при каждой отправке формы.
При успешной отправке формы отправляются **два события**:

1. **Общее событие** `form_submitted` — фиксирует любую заявку с сайта
2. **Уникальное событие** для конкретной формы

## Список событий для настройки

Необходимо создать следующие цели типа "JavaScript-событие" в интерфейсе Яндекс.Метрики:

### 1. Общая цель (для всех форм)

**Идентификатор:** `form_submitted`
**Название:** Отправка любой формы
**Описание:** Фиксирует отправку любой формы на сайте

---

### 2. Форма обратного звонка

**Идентификатор:** `callback_form_submitted`
**Название:** Заказ обратного звонка
**Описание:** Отправка формы "Заказать обратный звонок"

---

### 3. Форма получения расчета и макета

**Идентификатор:** `consultation_form_submitted`
**Название:** Получить расчёт и макет
**Описание:** Отправка формы "Получить расчёт и макет"

---

### 4. Форма задать вопрос

**Идентификатор:** `question_form_submitted`
**Название:** Задать вопрос
**Описание:** Отправка формы "Задать вопрос"

---

## Инструкция по созданию целей

1. Откройте настройки вашего счетчика Яндекс.Метрики (ID: **66944668**)
2. Перейдите в раздел **"Цели"**
3. Нажмите кнопку **"Добавить цель"**
4. Выберите тип цели: **"JavaScript-событие"**
5. Введите данные из списка выше:
   - **Идентификатор цели** — точное название события (например, `form_submitted`)
   - **Название** — описательное название для отчетов
6. Сохраните цель
7. Повторите для всех 4 событий

## Параметры событий

Каждое событие передает следующие параметры:

- `contactReason` — тип формы (`callback`, `consultation`, `question`)
- `formId` — ID формы
- `hasComment` — наличие комментария от пользователя (true/false)
- `hasProductOptions` — наличие выбранных опций продукта (true/false)

Эти параметры можно использовать для сегментации и детального анализа в отчетах Метрики.

```typescript
// Общее событие

  contactReason: "callback",
  formId: "callback",
  hasComment: true,
  hasProductOptions: false
});

// Уникальное событие формы
sendMetrikaGoal("callback_form_submitted", {
  contactReason: "callback",
  formId: "callback",
  hasComment: true,
  hasProductOptions: false
});
```

## Проверка работы

После настройки целей и деплоя:

1. Откройте сайт
2. Заполните и отправьте любую форму
3. В консоли браузера (в режиме разработки) вы увидите:
   ```
   [Yandex Metrika] Goal "form_submitted" sent
   [Yandex Metrika] Goal "callback_form_submitted" sent
   ```
4. В Яндекс.Метрике цели появятся в течение нескольких минут
