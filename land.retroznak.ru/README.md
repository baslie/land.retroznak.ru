# land.retroznak.ru

Лендинг компании «Ретрознак» версии 1.4.1 от 11 ноября 2025 года. Разработка велась с середины августа 2025 года. Сделан на Next.js 15 с TypeScript.

> **Примечание:** Этот проект является частью монорепозитория. См. [общий README](../README.md) для информации о всех проектах.

> **Продакшен:** Проект разворачивается на домен `land.retroznak.ru`. На одном домене работает только один проект (либо лендинг, либо конструктор).

```bash
# Установка зависимостей
npm install
```
